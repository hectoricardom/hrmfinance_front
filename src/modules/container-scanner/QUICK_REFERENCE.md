# Container Scanner - Quick Reference Card

## 🚀 How to Use

### Step 1: Scan Container
```
1. Open the Scanner app
2. Point camera at container QR code
3. Wait for green checkmark
```

### Step 2: Scan Bulks
```
1. Scan each bulk QR code
2. Watch progress bar fill up
3. Green checkmark = success
4. Yellow warning = already scanned
```

### Step 3: Finish
```
1. When all bulks scanned:
2. Tap "Mark as Received"
3. See success message
4. Ready for next container
```

## ✅ What Means What

| Symbol | Meaning |
|--------|---------|
| 📦 | Scan container QR code |
| 📱 | Scan bulk QR code |
| ✓ | Successfully scanned |
| ⚠️ | Warning or error |
| 🎉 | All done! |

## 🔴 If Something Goes Wrong

### Error: "Container not found"
**Fix:** Check QR code, try scanning again

### Error: "Bulk not found"
**Fix:** This bulk is not in this container

### Warning: "Already scanned"
**Fix:** This bulk was already scanned, skip it

### App not working
**Fix:** Close app, restart, try again

## 💡 Tips

- ✓ Hold phone steady when scanning
- ✓ Make sure QR code is well-lit
- ✓ Scan from 6-12 inches away
- ✓ If stuck, tap "Cancel" and start over
